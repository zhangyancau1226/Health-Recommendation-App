CS 125 Project: Calorie Calculator
Our app records restaurants data of three UCI food courts and users can select what they eat so our app can calculate their daily calorie input and give them health advice.

Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

Installing
First, open your command line and change directory (cd) to the project folder
Prerequisites£º
1.	Node.js
2.	Expo
3.	One cell phone simulator. e.g. android studio

Installing:
Type:
	$ npm install
	$ npm start
If the start fails, it means you miss some necessary libraries.
Type:
	$ npm install missing
	Then restart the project
	$ npm start


Directory explanation:
API directory contains all the APIs we made.
"assets" contains all the pictures we used in our project
"components" contains all the styles of javascripts
"navigation" contains all the navigation code for the page nagivation in our project.
"screens" contains all the front-end page of our project.
"App.js" is our root file
"styles.js" is the style javascript which makes our project beautiful.
